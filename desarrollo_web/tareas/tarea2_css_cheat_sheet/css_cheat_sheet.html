Border

La propiedad border crear un borde del mismo grosor en todas direcciones sobre los elementos que se encuentren dentro de él.

Elemento con bordes standar

Border-left, Border-right, Border-top, Border-bottom

Permite definir el estilo y grosor de cada borde imdependiente del bloque.

Izquierda - Rojo
Derecha - Azul
Arriba - Verde
Abajo - Negro

Border-radious

Permite redondear los bordes de un elemento. Puede aplicarse a todas o tener un valor independiente por borde.

Elemento con bordes redondeados

Background-image

Permite insertar una imagen de fondo dentro del bloque.

Display

Establece el tipo de visualización de un conjunto de elementos en pantalla.

flex

Ordena los elementos en fila o columnas formando un bloque con la posibilidad de alterar el orden que llevan (row-reverse en este caso).

grid

Crea una maya en la que le reserva un espacio a cada elemento y se puede manipular su distribución entre filas y columnas.

Position

Determina la posición final del elemento dentro de un bloque sin que los elementos que no son modificados no buscan llenar el espacio que se crea.

Filter

Como si se tratase de la edición de una imagen, esta propiedad aplica efectos gráficos que cambian la apariencia del elemento.
