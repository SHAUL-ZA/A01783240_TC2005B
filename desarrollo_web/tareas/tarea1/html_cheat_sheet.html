Uso de los Tags en HTML

Volver

En este documento se muestran los tags mas usados en HTML

Los tags se pueden usar de dos formas:

  • Con etiqueta de apertura y cierre
  • Con etiqueta de apertura

Los tags de apertura y cierre se usan para agrupar contenido

Los tags de apertura se usan para agregar contenido, como se muestra en los siguientes ejemplos:

  • El archivo html está nombrado con clases y otros elementos que pueden ser utilizados con css en caso de que se le de continuidad a este trabajo y se le aplique una hoja de estilos

Tag a

Etiqueta utilizada para la creación de hipervíncluos donde podemos insertar referencias de enlaces a otra pagina web. Se utiliza "href" y se le da el atributo, después entre la apertura y cierre se ingresa el nombre con el que se presentará el vínculo.

Ejemplo:

<a href="https://www.google.com/"> Liga a Google </a >

Liga a Google

Tag p

Etiqueta designada para parrafos que generalmente se utilizan para agregar información sobre algún elemento de la página.

Ejemplo:

<p> Esta es una etiqueta de parrafo dentro de la tarea html_cheat_sheet </p>

Esta es una etiqueta de parrafo dentro de la tarea html_cheat_sheet

Tag h1, h2,...h6

Etiquetas de encabezados. Poseen una jerarquía de h1 hasta h6 y se ocupan depenediendo el caso Unicamente puede haber un h1 por página que se ocupa como encabezado principal.

Ejemplo:

<h1> Encabezado principal </h1>
<h2> Encabezado secundario </h2>
<h3> Encabezado h3 </h3>
<h4> Encabezado h4 </h4>
<h5> Encabezado h5 </h5>
<h6> Encabezado h6 </h6>

Encabezado secundario

Encabezado h3

Encabezado h4

Encabezado h5

Tag div

Los div se ocupan para dividir un grupo de elementos de la pagina que se ocupan como contenedores para dar estilos a esos elementos con significado específico con ayuda de CSS.

Ejemplo:

Obtenido de un frontend de una tienda de camisetas hice hace unos meses:
<div class="producto">
    <a href="#">
         <img class="producto__imagen" src="#" alt="imagen camisa">
         <div class="producto__informacion">
             <p class="producto__nombre">Github</p>
             <p class="producto__precio">$25</p>
             </div>
         </a>
     </div> <!--.producto-->

Github

$25

Tag span

La etiqueta span se ocupa en elementos de nivel de linea que dentro de ella se puede encerrar el contenido al que se le quiere aplicar un atributo o estilo específico pero que no deja de pertenecer a su etiqueta general.

Ejemplo:

<p> Como se menciona en la tarea, <span style="color: rgb(230, 12, 12);"> span </span> es una etiqueta que se ocupa para darle estilo a un texto específico dentro de un párrafo. </p>

Como se menciona en la tarea, span es una etiqueta que se ocupa para darle estilo a un texto específico dentro de un párrafo.

Tag strong

La etiqueta strong es utilizada para poner en negrita el texto que esté dentro de su espacio. strong posee un significado semántico que busca denotar importancia al texto.

Ejemplo:

<p> <strong> strong </strong> es una etiqueta que se ocupa para darle estilo a un texto específico dentro de un párrafo. </p>

strong es una etiqueta que se ocupa para darle estilo a un texto específico dentro de un párrafo.

Tag i

La etiqueta i se ocupa de manera semántica para transmitir texto a voz o tono alternativo.

Ejemplo:

<p> Los <i> bits </i> son los elementos básicos de la informática. </p>

Los bits son los elementos básicos de la informática.

Tag u

La etiqueta u se ocupa para poner en subrayado el texto que esté dentro de su espacio.

Ejemplo:

<p> Los <u> bits </u> son los elementos básicos de la informática. </p>

Los bits son los elementos básicos de la informática.

Tag b

La etiqueta b se ocupa para poner en negrita el texto que esté dentro de su espacio. b no posee un significado semántico que busca denotar importancia al texto.

Ejemplo:

<p> Esto es un texto normal <b> Esto es un texto en negritas</b> </p>

Esto es un texto normal Esto es un texto en negritas

Tag strike

La etiqueta strike se ocupa para poner en tachado el texto que esté dentro de su espacio.

Ejemplo:

<p> Esto es un texto normal <strike> Esto es un texto tachado</strike> </p>

Esto es un texto normal Esto es un texto tachado

Tag small

La etiqueta small se ocupa para poner en pequeño el texto que esté dentro de su espacio. Busca semánticamente transmitir información secundaria como derechos de autor, fecha de publicación, etc.

Ejemplo:

<small> tarea publicada el 10 de abril de 2023 </small>

tarea publicada el 10 de abril de 2023

Tag em

La etiqueta em se ocupa para poner en cursiva el texto que esté dentro de su espacio. em posee un significado semántico que busca denotar importancia al texto.

Ejemplo:

<p> Le das mucha <em> importancia</em> al asunto. </p>

Le das mucha importancia al asunto.

Tag header

La etiqueta header se ocupa para agrupar elementos de encabezado de la página que definen el encabezado.

Ejemplo:

<header>
<h2> Título </h2>
<h3> Subtítulo </h3>
</header>

Título

Subtítulo

Tag section

La etiqueta section se ocupa para identificar un grupo de elementos con una temática de contenido específica. También se utiliza para dividir el contenido de la página en secciones.

Ejemplo:

<section>
<h2> datos de contacto </h2>
<h3> numero </h3>
<a> mail <a>
</section>

datos de contacto

numero

mail

Tag article

La etiqueta article se ocupa para identificar un grupo de elementos que forman un contenido independiente. Por ejemplo, un artículo de un blog, un mensaje de un foro, etc. Si una etiqueta de article se elimina, el contenido que contiene no pierde sentido.

Ejemplo:

<article>
<h3> HTML </h3>
<p> HTML es un lenguaje utilizado para la elaboración de páginas web </p>
</article>
<article>
<h3> CSS </h3>
<p> CSS es un lenguaje utilizado para la creación de estilos de páginas web </p>
</article>

HTML

HTML es un lenguaje utilizado para la elaboración de páginas web

CSS

CSS es un lenguaje utilizado para la creación de estilos de páginas web

Tag aside

La etiqueta aside se ocupa para poner contenido indirectamente relacionado con el contenido principal de la página. Se utiliza para poner contenido secundario como barras laterales, publicidad, etc.

Ejemplo:

Fui a visitar la ciudad de Tokio en Japón.

<aside>
<h3> Información sobre Tokio </h3>
<p> Tokio es la capital de Japón y la ciudad más poblada del mundo. </p>
</aside>

Información sobre Tokio

Tokio es la capital de Japón y la ciudad más poblada del mundo.

Tag footer

La etiqueta footer se ocupa para agrupar elementos que definen el final de la página.

Ejemplo:

<footer>
<p> © 2023 - Todos los derechos reservados </p>
</footer>

© 2023 - Todos los derechos reservados

Tag nav

La etiqueta nav se ocupa para agrupar los principales enlaces de navegación de la página.

Ejemplo:

<nav>
<a href="#"> Inicio </a>
<a href="#"> Preguntas Frecuentes </a>
<a href="#"> Sobre Nosotros </a>
<a href="#"> Contactanos </a>
</nav>

Inicio Preguntas Frecuentes Sobre Nosotros Contactanos

Tag img

La etiqueta img se ocupa para insertar una imagen en la página.

Ejemplo:

<img src="img/teamwork.jpg" alt="Trabajo en equipo">

Tag br

La etiqueta br se ocupa para hacer un salto de línea en el texto. br no requiere de etiqueta de cierre.

Ejemplo:

<p> Hola <br> ¿Cómo estás? </p>

Hola
¿Cómo estás?

Tag hr

La etiqueta hr se ocupa para hacer una línea horizontal en el texto para separar el contenido.

Ejemplo:

HTML es un lenguaje utilizado para la elaboración de páginas web.

CSS es un lenguaje utilizado para la creación de estilos de páginas web.

Tag form

La etiqueta form se ocupa para agrupar elementos de un formulario donde se puede ingresar información y esta ser enviada. Se emplea para crear formularios de contacto, encuestas, etc.

Ejemplo:

Contactános llenando todos los campos

Tag table

La etiqueta table se ocupa para crear una tabla en la página. table posee una estructura de etiquetas que se deben respetar para que la tabla se muestre correctamente como tr y td.

Ejemplo:

Nombre Apellido Edad
Shaul Zayat 20
Victor Hernandez 21
Guillermo Sanchez 25

Tag ul

La etiqueta ul se ocupa para crear una lista desordenada en la página. ul posee una estructura de etiquetas que se deben respetar para que la lista se muestre correctamente como li.

Ejemplo:

  • Frappe
  • Expresso
  • Té

Tag ol

La etiqueta ol se ocupa para crear una lista ordenada en la página. ol posee una estructura de etiquetas que se deben respetar para que la lista se muestre correctamente como li.

Ejemplo:

  1. Frappe
  2. Expresso
  3. Té
